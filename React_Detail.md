
# __React__
* ##### React 개요
  * Vitual Dom 기반 : 변화만 필요한 부분만 다시그리는 형식
  * WebPack : JS 를 합치고 빌드 해주는것
  * Babel : (JavaScript Compiler) 모던 JS 를 구버전으로 변환 해주는것
  * JSX : React를 위한 자바 스크립트
